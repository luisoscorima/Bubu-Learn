1. Funciones de Excel

a) SUMA

=SUMA(A1:A5) suma los valores de un rango.

b) SI

=SI(A1>10, "Mayor", "Menor o igual") verifica condiciones.

c) BUSCARV

=BUSCARV("Luis", A2:B10, 2, FALSO) busca datos en tabla vertical.

Ejemplo: Dibujar un cuadrado

Salida:

Más ejemplos para probar:

Círculo:

import turtle
t = turtle.Turtle()
t.circle(100)
turtle.done()

Triángulo:

import turtle
t = turtle.Turtle()
for i in range(3):
    t.forward(100)
    t.left(120)
turtle.done()
